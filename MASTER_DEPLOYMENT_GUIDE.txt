IF SOMETHING GOES WRONG
================================================================================

Script Fails to Run:
  • Right-click on .bat file → "Run as Administrator"
  • Make sure Git is installed: https://git-scm.com
  • Restart Command Prompt and try again

Build Still Fails:
  • Check: https://vercel.com/dashboard
  • Look for error details in Vercel logs
  • Check GitHub for committed changes: git log -3

Cart Still Broken:
  • Check browser console (F12) for JavaScript errors
  • Clear browser cache (Ctrl+Shift+Delete)
  • Try incognito window

Can't Push to GitHub:
  • Check: git status (should be clean)
  • Try: git config user.name "Your Name"
  • Verify Git is installed
